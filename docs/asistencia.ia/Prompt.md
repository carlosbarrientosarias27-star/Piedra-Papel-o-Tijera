# Documentación de Asistencia con IA

## Proyecto
Juego Piedra, Papel o Tijera en Python

## Prompts utilizados

### Prompt 1
"Crear un juego de piedra, papel o tijera en Python que permita múltiples rondas"

### Prompt 2
"Agregar contador de victorias, derrotas y empates"

### Prompt 3
"Calcular y mostrar el porcentaje de victorias del jugador"

### Prompt 4
"Mostrar un resumen completo al final de la partida"

### Prompt 5
"Determinar y mostrar el ganador final de la partida"

### Prompt 6
"Mejorar la interfaz visual del juego en consola usando emojis y formato atractivo"

## Mejoras implementadas gracias a IA

- Validación de entradas del usuario
- Sistema de rondas múltiples
- Contador acumulado de resultados
- Cálculo automático del porcentaje de victorias
- Determinación del ganador final
- Interfaz mejorada con emojis
- Limpieza de pantalla para mejor experiencia

## Herramienta utilizada
ChatGPT (modelo de lenguaje basado en IA)
